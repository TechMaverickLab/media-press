First setup

To use Media Press, you need to select a folder on your computer. The program will automatically create a "Media Press Files" subfolder in it, where the originals (`source`) and the processing results (`output`) will be stored.

We recommend that you select the "Documents" folder or a folder on your disk with sufficient free space.
